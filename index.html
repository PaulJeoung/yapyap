Java

- 자바 기본 문법 숙지
- List, ArrayList, Iterator, Map, HashMap
- Exception, Thread, Network 기본 개념
- 데이터베이스 연동 실습

Spring Framework

- 스프링 프레임워크 기초 숙지
- IoC, DI, AOP 등의 스프링 핵심 컨셉 숙지
- MVC, PRG 패턴 사용
- CRUD를 통한 DB와의 연동

MySQL (+H2)

- 테이블의 생성, 수정, 삭제, 검색 등 기본적인 쿼리문 작성

Javascript

- Javascript를 사용하여 웹 페이지의 동적인 기능을 구현
- 기본적인 문법 숙지 및 function을 이용한 이벤트 처리 숙지

React

- 컴포넌트 기반 개발 방식으로 UI를 독립적인 컴포넌트로 나누어 개발하고 재사용
- JSX 을 사용하여 HTML과 유사한 구문으로 컴포넌트를 작성
- Lifecycle method 숙지
- state, props, hooks, redux, router, Virtual DOM 숙지

HTML5

- 웹 페이지의 기본 구조를 설정하고, 다양한 태그를 사용하여 웹 페이지를 구성
- 기본 문법 숙지
- Table 태그, form 태그, li 태그, a 태그

CSS3

- CSS를 사용하여 웹 페이지의 텍스트, 배경, 레이아웃 등을 스타일
- Flexbox와 Grid를 사용한 레이아웃 설계
- 미디어 쿼리를 이용한 반응형 설계 (@media 사용)
- @keyframes 와 transition 속성을 이용한 애니메이션 전환 효과
- Bootstrap을 이용한 CSS 프레임워크로 빠르고 일관된 디자인으로 구현

팀프로젝트 - 럭셔리 스니커즈 쇼핑몰

6개월 간의 KDT 직업 훈련과정인 RPA, 시큐어코딩 활용 자바스프링의 수업 후 팀프로젝트를 2024년 12월 23일 부터 2025년 1월 17일까지 약 4주간 진행 했습니다. 주제는 워너비 명품 쇼핑몰 구축 개발이었고, 주제에 맞는 컨셉으로 리셀이 가능한 럭셔리 스니커즈 쇼핑 플랫폼을 선정 하게 되었습니다.

이 프로젝트에서는 저는 조장을 맡아 프로젝트 전반에 대한 부분을 관리와 일부 모듈을 개발에 참여 하였습니다.

제가 맡은 모듈은 백엔드는 상품기능 구현, 결제 모듈 구현, 배송추적 모듈 구현, 게시판 기능 구현을 맡았고, 프론트엔드는 게시판화면과 AboutUs 화면과 함께 관리자페이지를 개발 했습니다.

조장의 역할로는 팀원 업무 분배 및 Github 소스코드 충돌시 리팩토링, 프로젝트 전반에 대한 문제확인, 제작 발표를 위한 PPT작업 및 비디오 영상 제작등을 맡았고 노션의 "쇼핑몰 구축 프로젝트"에 접속 하시면 시연영상과 함께 저의 구현과정을 자세히 살펴 보실 수 있습니다
